# WWTP AI Safety Factors

This repository contains AI-evaluated updates to safety factors in wastewater treatment plant (WWTP) design, based on literature and raw data from California plants. It includes methodologies, simulation codes, and data.

## Setup
- Install dependencies: `pip install -r requirements.txt`
- Run models: e.g., `python hydraulic_model.py`

## Files
- methodology.md: Updated AI evaluation methodology.
- data.csv: Raw data on flows, peaking factors, BOD limits.
- *_model.py: Simulation scripts for each aspect.

For details, see methodology.md.
